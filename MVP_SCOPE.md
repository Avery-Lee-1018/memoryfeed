# MVP Scope

## Must Have
- 소스 등록(벌크 URL 입력)
- 소스 목록/활성화/삭제
- 날짜별 3개 피드
- 날짜 이동 + TODAY 복귀
- 카드 대체(`오늘은 안볼래요`)
- 메모 저장/조회/삭제
- D1 저장 + Worker API

## Nice to Have
- 소스 품질별 수집 안정화
- 에러 피드백 세분화
- 로딩/전환 인터랙션 고도화

## Not Now
- 추천 알고리즘 고도화
- 태그/폴더 기반 정리 UI
- 멀티 디바이스 동기화 고도화
- 협업 기능
- 정교한 관리자 대시보드

## UX Constraint
- 카드 정보 3줄 이하
- 버튼 최소화
- 스크롤 최소화

# MVP Scope

## Must Have
- 소스 등록 (RSS 피드 or 블로그 URL) — 등록 1회, 이후 수집 자동
- 콘텐츠 수집
- 하루 3개 피드
- 스와이프 인터랙션
- 반응 저장 (like / snooze / remember)
- 랜덤 셔플
- D1 저장
- Cron daily refresh

## Nice to Have
- 간단 요약
- 가벼운 질문 1개

## Not Now
- 추천 알고리즘 고도화
- 태그 기반 필터
- 소스 추천
- 노션 연동
- 멀티 디바이스 동기화
- 협업

## UX Constraint
- 카드 정보 3줄 이하
- 버튼 최소화
- 스크롤 최소화

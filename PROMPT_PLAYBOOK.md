# Prompt Playbook

## Claude 역할
- 제품 원칙 유지
- 과설계 방지
- UX 감수
- 다음 작업 정의

## Codex 역할
- 코드 생성
- 파일 수정
- 구조 구현

---

## Claude Prompt Template

현재 상태:
(간단 요약 5줄 이하)

검토 기준:
- 과설계 여부
- 제품 원칙 위반 여부
- UX 단순성

요청:
1. 제거할 것
2. 유지할 것
3. 단순화할 것
4. 다음 작업 3개

---

## Codex Prompt Template

목표:
(한 문장)

현재 상태:
(파일 or 기능 설명)

요청:
- 이 파일만 수정
- 최소 구현
- 과한 구조 금지

산출물:
- 수정된 코드
- 변경 파일 목록

---

## Rules

- 긴 설명 금지
- 한 번에 하나의 작업
- 항상 md 기준 참고
- 결과 → Claude 검토 → 다시 Codex

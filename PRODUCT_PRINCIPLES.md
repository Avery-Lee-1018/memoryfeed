# Product Principles

1. **저장이 아니라 재등장** — 북마크가 아니다. 언젠가 다시 떠오르게 하는 것이 전부다.
2. **하루 3개만** — 그 이상은 부담이다. 덜 보여줄수록 더 본다.
3. **정리 없이 흘려라** — 폴더, 카테고리, 강한 태그 없음. 던지면 끝.
4. **느낌 좋은 셔플** — 정확한 추천도 완전 랜덤도 아니다. 설명 안 되는 우연처럼.
5. **카드 1장이 전부** — 스크롤 없음. 관리 UI 없음. 지금 이 카드만.
6. **스와이프가 유일한 반응** — 클릭보다 가볍고, 관리보다 반응.
7. **질문은 살짝만** — 깊은 사고 강요 없음. 스치는 트리거면 충분.
8. **언제나 열 수 있어야 한다** — 밀린 게 없고, 정리 안 해도 된다.
9. **백엔드 단순, 경험은 프론트에서** — API 4개면 충분하다.
10. **과설계는 즉시 롤백** — 기능 추가 전 "이게 정말 필요한가?" 먼저.
